## 🔧 Quiz Application

### Overview

This project is a fully interactive **Quiz Application** built using **HTML**, **CSS**, and **JavaScript**. The app presents users with a series of multiple-choice questions, tracks their progress, and displays a final score at the end. It includes dynamic transitions, a sleek design, and responsive features for an engaging user experience.

### 🔑 Features

- **Dynamic Quiz Flow**: Questions are presented one at a time, with immediate feedback for correct and incorrect answers.
- **Progress Tracking**: A progress bar updates dynamically as users answer questions.
- **Score Display**: Final score is displayed at the end of the quiz.
- **Restart Option**: Users can restart the quiz to try again.
- **Smooth Transitions**: Seamless UI transitions between questions and results.

---

### 🛠️ Technologies Used

- **HTML**: Structure and content of the quiz.
- **CSS**: Styling for a modern, user-friendly interface.
- **JavaScript**: Logic for question navigation, answer validation, and score calculation.

---

### ⚖️ How It Works

1. **Initial Load**: The app displays the first question along with answer options.
2. **Answering Questions**:
   - Users select an option.
   - Correct and incorrect answers are highlighted.
   - The "Next" button becomes active.
3. **Progress Update**: The progress bar visually reflects the user's progress through the quiz.
4. **Final Score**: At the end of the quiz, the user's total score is displayed.
5. **Restart**: Users can restart the quiz to attempt it again.
